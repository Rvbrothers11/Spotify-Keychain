# Spotify-Keychain
A 3D printed and PCB designed Spotify code integrated keychain!

I wanted to create these keychains and hang them on my bags. One on my school bag and the other on my badminton bag. 

I created the 3D printed keychain first:
<img width="844" height="293" alt="Screenshot 2026-05-17 174527" src="https://github.com/user-attachments/assets/cbf099cd-9ab9-4131-9427-e94f88f386f7" />

And then moved on to create the PCB design:
<img width="986" height="301" alt="Screenshot 2026-05-19 175347" src="https://github.com/user-attachments/assets/61f60b3f-8934-4c9c-b45d-c18410b028e2" />

How to use:
Just 3D print or order the PCB files which have pre-cut holes for the keychain. Add your own keychain and connect it to your bag. Take your phone, open spotify and use the code scanner to play the song.
